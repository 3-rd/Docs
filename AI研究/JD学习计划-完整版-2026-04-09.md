## AI Native Infra 全景图（补充知识）

来自 AI Native Infra 概览文档，AI Infra 技术栈分层如下：

```
┌─────────────────────────────────────┐
│  AI Gateway（AI 网关）               │  ← 多模型路由、流量治理、token 计费
├─────────────────────────────────────┤
│  AI Agent 应用层                    │  ← LangChain / LangGraph / CrewAI
├─────────────────────────────────────┤
│  Agent Memory / Context 管理        │  ← 4类记忆、Context Engineering
├─────────────────────────────────────┤
│  向量数据库（Vector DB）             │  ← Milvus / Faiss / Pinecone
├─────────────────────────────────────┤
│  LLM 推理引擎                       │  ← vLLM / SGLang / TGI
└─────────────────────────────────────┘
```

**关键认知**：AI Infra 不是孤立的"底层"，而是一个从芯片到应用的完整栈。理解每一层的作用和交互方式，比单独掌握某层更重要。
