# AI 转行学习计划

## 📅 时间安排

| 时段 | 可用时间 |
|------|----------|
| 工作日晚上 | 2-3 小时 |
| 周末 | 4-6 小时/天 |

---

## 🎯 阶段一：摸清方向（1-2周）

**目标**：了解 AI 行业有哪些岗位、具体做什么，找到最适合自己的方向

**任务**：
- [ ] 调研 AI 行业主要岗位（LLM infra、Agent 开发、RAG 开发、AI 产品经理等）
- [ ] 了解各岗位技能要求、薪资范围、发展前景
- [ ] 与老板深度沟通，确定方向

**具体内容**：
1. **LLM Infra**：模型部署、微调、推理优化、GPU 集群管理
2. **Agent 开发**：LangChain、AutoGen、Agent 框架、Prompt 工程
3. **RAG**：向量数据库、检索系统、知识库
4. **AI 应用开发**：API 封装、产品集成

---

## 🎯 阶段二：基础夯实（2-4周）

确定方向后，补基础技能

**可能涉及的**：
- Python 深入（协程、异步、装饰器）
- LLM API 调用（OpenAI、Claude、国产模型）
- 向量数据库（Milvus、Qdrant、Chroma）
- Docker / K8s 基础
- GitOps 基础

---

## 🎯 阶段三：项目实战（4-8周）

**目标**：产出 1-2 个拿得出手的项目

**方向示例**：
- Agent 方向：智能客服 Agent、个人助理 Agent
- Infra 方向：LLM 部署 Demo、推理服务化
- RAG 方向：私有知识库问答系统

---

## 🎯 阶段四：求职准备（2-4周）

- 简历优化
- 面经准备
- 模拟面试

---

## 📬 主动推送

- 每周五：本周学习总结 + 下周计划
- 每天早上：行业新闻/技术干货（可选）
- 有新机会/动态时：及时推送

---

**注意**：这是初步框架，确定方向后会细化~
